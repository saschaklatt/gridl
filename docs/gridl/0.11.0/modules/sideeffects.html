forEachCell

  • forEachCell<T>(grid: Grid<T>, callback: ForEachCellCallback<T>, walk?: GridWalker): void
    • Defined in sideEffects/forEachCell.ts:42

    Calls the given callback function on each cell in the grid.

    example
    import {forEachCell} from "gridl/sideEffects";

const grid = createGridFromArray2D([
    [1, 2, 3],
    [4, 5, 6],
]);
let str = "";
forEachCell(grid, (cellValue) => {
    str = `${str}${cellValue}`;
});
// => str === "123456"

    Type parameters

    • T

      The cell type.

    Parameters

    • grid: Grid<T>

      The grid in which to search.

    • callback: ForEachCellCallback<T>

      The callback function that is called on each cell.

    • Optional walk: GridWalker

      This function calculates the grid position based on the iteration step and therefore defines the order in which the cells are iterated.

    Returns void
